# Talk-to-Fly
The integration of Large Language Models (LLMs) with Unmanned Aerial Vehicles (UAVs) is a significant paradigm shift that enables increased accessibility, and control over traditional manual piloting or waypoint missions.
This paper introduces the Talk-to-Fly system, which enables a user to issue complex natural language commands, which are interpreted and executed as UAV actions.
We propose a memory mechanism for increased environmental understanding, and develop a validation layer for improved safety.
The system underwent testing across various levels of command complexity in simulation and real-world experiments.
Preliminary results show high levels of responsiveness and accuracy in task completion.
The results suggest that this approach is a transformative step toward making advanced UAV control more accessible and efficient.
